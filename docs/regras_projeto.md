# Regras do Projeto

Este arquivo documenta as preferências e regras globais definidas pelo usuário para o desenvolvimento deste projeto.

## Preferências Gerais
- **Idioma:** Sempre responder e documentar em Português do Brasil (PT-BR).
- **Documentação:** Manter a pasta `docs/` atualizada.
- **Autoria:** Documentar autoria no código quando solicitado.
- **Desenvolvimento:** Seguir boas práticas, otimização e Clean Code.
